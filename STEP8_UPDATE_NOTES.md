# Step 8 UI Update Notes

This version improves the Step 8 weekday quick-select feature.

## Added

- Larger quick-select buttons for month-level weekday selection.
- Korean / English language toggle.
- Bigger mobile-friendly quick-select grid.
- Quick-select changes now update the calendar immediately before Firestore sync completes.
- Quick-select buttons show a selected state when all matching dates are already selected.

## Important

Copy your existing Firebase config into `script.js` before deploying.
