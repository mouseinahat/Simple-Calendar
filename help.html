Simple Calendar

도움말

초대받은 방에 들어가고, 프로필을 선택한 뒤 가능한 날짜를 표시하는 순서입니다.

←

초대받은 경우

  1. 공유받은 링크를 열면 방 비밀번호 입력이 먼저 보입니다.
  2. 비밀번호를 입력해 방을 열면 프로필 목록이 먼저 표시됩니다.
  3. 이름을 검색하거나 목록에서 내 프로필을 선택합니다.
  4. 프로필 비밀번호를 입력하면 달력을 수정할 수 있습니다.

날짜 표시

  1. 달력 날짜를 누를 때마다 가능, 미정, 불가 순서로 바뀝니다.
  2. 현재 열려 있는 프로필로 표시한 날짜는 날짜 칸 안에서 강조됩니다.
  3. 추천 날짜는 많은 사람이 가능한 날짜를 기준으로 정렬됩니다.
  4. 확정 버튼을 다시 누르면 확정을 취소할 수 있습니다.

새 방 또는 새 프로필

  1. 새 방 만들기는 초대받은 방 입장보다 낮은 위치에 있습니다.
  2. 새 프로필 만들기는 프로필 선택 화면의 버튼을 눌러 펼칩니다.
  3. 프로필 색상은 아직 쓰이지 않은 색상 중 하나가 자동으로 선택됩니다.
